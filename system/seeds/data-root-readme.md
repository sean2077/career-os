# Career Home

> [!info] User-owned data
> This directory is the user-owned data layer for this Career Home instance.
> System updates must never overwrite these files.

Describe the outcome you want to an Agent. Career OS selects and composes the
appropriate Skills, writes through the owning authority, and leaves canonical
records here for review in Obsidian.

## Ownership and Privacy

Records and filenames may use any Unicode language, with optional BCP 47
metadata where the record schema supports it. This data root is eligible for
the user's Git history.

The Career OS MIT License covers the upstream system, tooling, schemas, and
public documentation. It does **not** grant permission to copy, publish,
redistribute, or reuse the personal data, identity material, resume content,
attachments, or private judgments stored in this data root. An item is
redistributable only when its owner grants a separate explicit license or
publication authorization for that item.

Machine-readable source inventories and provenance maps belong under
`.provenance/`. They are control data, not Markdown career records.
