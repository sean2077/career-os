# Optional QuickAdd Adapter

`capture-choice.json` is a generated, reviewable QuickAdd `2.12.3` Capture
choice. It appends raw text to the Career Evidence inbox; it does not create an
approved claim or a canonical work record.

Career OS validates the installed plugin version and checks for choice ID/name
conflicts, but does not edit `.obsidian/plugins/quickadd/data.json`. Import or
recreate the choice explicitly in QuickAdd after reviewing the generated JSON.
