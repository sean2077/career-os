# Security and Privacy

Career OS is local-first and stores no account credentials. Keep personal records in a private repository or outside version control as appropriate.

Report a vulnerability through a private maintainer channel rather than a public issue. Do not attach real career records, resumes, access tokens, or other personal data to a report.

The CLI never sends applications, messages, uploads, or account changes. Public and application-grade exports require an explicit command and pass deterministic privacy checks before a PDF is copied to its destination.
