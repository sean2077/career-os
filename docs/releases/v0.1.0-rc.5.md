## Operating-model boundary

The recommended personal installation is now an independent private sibling
repository mounted into an existing Vault through a host-tracked relative
directory symlink. The public repository is a fetch-only `upstream` with an
explicit `DISABLED` push URL; an optional `origin` is usable only after the
owner confirms that its hosted visibility is private. Career OS validates this
local configuration without contacting or mutating a remote.

An external project root is accepted only through one configured
Vault-relative POSIX mount whose final component is a real directory symlink to
that exact project. Absolute, drive-qualified, traversal, backslash, dangling,
plain-directory, wrong-target, and untracked host mount states fail or warn as
documented. Attach and detach never create, ignore, rewrite, or remove the
host-owned link.

## Installation and Obsidian boundary

The English installation guide now separates core, live Obsidian, resume, and
optional PDF-inspection readiness. Font binaries remain outside Git; the
tracked manifest pins two Noto Sans CJK SC 2.004 files totaling 33,439,612 bytes
by URL, size, and SHA-256. A fresh offline clone can use the core system but
cannot build resumes until those files have been fetched and verified.

The public Base adds a Same-company application decision view. It groups
reviewed, non-rejected, evidence-fit screenings by typed `recruiting-scope`
references; absent scope references remain visibly unresolved rather than
being inferred from a display name.

## Publication contract

The public repository is
[`sean2077/career-os`](https://github.com/sean2077/career-os). Publication is
not complete when `main` or the tag is merely pushed. The Release workflow must
check out the tag commit, validate the exact changelog section, re-run the core,
Skill, harness, resume build, and privacy-export gates, and create a non-draft
GitHub prerelease that identifies the same tag. The hosted workflow run and
Release URL are verified after the immutable tag is pushed.
