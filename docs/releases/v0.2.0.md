## Public extraction

The contribution was produced from an allowlisted public-path diff; no Home Git
history was merged or cherry-picked. Public adaptation retains
`development_topology = "standalone-framework"`, the default system font
directory, and disabled OpenCLI configuration. Personal career records,
identity, attachments, local fonts, `.obsidian/`, runtime, build, and
`.career-os/` state are excluded.

The existing extraction supplement now hash-binds both changed files and
reviewed deletions. Changed guarded resume/privacy tests and the privacy
implementation were manually reviewed as synthetic-only before their hashes
were added to `system/privacy/public-fixture-policy.json`.
